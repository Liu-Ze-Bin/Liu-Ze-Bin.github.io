如何创建Linux端GitHub博客

七月 01, 2019
  • -GitHub -博客 -Linux

1、官网下载nodejs Linux系统的长期支持版本
http://nodejs.cn/

2、双击 下一步 安装nodejs

3、切换到root权限
node -v
npm -v 查看版本信息

—–（以下命令是在获取root权限下操作的 不是必须要root权限 可以自定义）—–

4、使用npm安装cnpm root权限下输入以下命令：

npm install -g cnpm --registry=https://registry.npm.taobao.org

5、使用cnpm全局安装hexo（未退出root权限）

cnpm install -g hexo-cli

安装完成后使用hexo -v查看版本 有版本信息表示安装成功

6、新建一个文件夹 mkdir hexo（后期安装有错误可以直接删除该文件夹重新操作）
进入文件夹 cd hexo
初始化 sudo hexo init
最后一行显示 “INFO Start blogging with Hexo!”表示安装成功

7、启动博客
命令：hexo s
如下提示：
—>INFO Start processing
—> INFO Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.

Linux下打开浏览器输入http://localhost:4000或127.0.0.1:4000

--可以在本地启动了--

8、新建一篇文章
ctrl+c断开连接
命令：hexo n(new) “输入要创建的文章名称”；在下面提示路径中可以查看文件已经创建
—>INFO Created: /home/Wen/hexo/source/posts/My-hexo-blog.md
可以使用vi编辑器直接打开添加内容
9、生成
编辑内容保存后 退出到创建博客文件夹的（我的是hexo）目录下
清理一下 hexo clean （按需使用）
生成 hexo g
启动查看 hexo s --新建的文章已经有了-_-

10、部署到Github上（自行注册Github账号）
登录到个人Github账号—–>新建一个仓库
注意命名方式需使用 账号昵称.github.io

在hexo目录下安装部署插件：
输入cnpm install --save hexo-deployer-git

配置_config.yml文件
vi打开_config.yml文件  最后三行改成如下
type: git
repo: https://github.com/Monk-J0/Monk-J0.github.io.git
branch: master  
保存退出

11、部署到远端
运行
git config –global user.email “you@example.com“输入自己的github认证邮箱
git config –global user.name “Your Name”输入自己的github账号

 命令：hexo d(deploy部署)
 --->INFO  Deploy done: git  -_-部署完成,在浏览器里点击刷新，仓库已经有博客内容了-_-

直接在浏览器中直接输入仓库名（我的是Monk-J0.github.io）就可以查看部署到github上的内容

12、主题DIY
在hexo的官网可以找到更多的主题，选择一个自己喜欢的，然后找到对应的GitHub库，点下载或克隆，复制里面的链接 

   hexo目录下 输入命令下载 并克隆到themes目录下
   git clone 链接(如https://github.com/litten/hexo-theme-yilia.git) themes/主题名字(如yilia)

  再次修改hexo目录下_config.yml文件
  vi  _config.yml 打开配置文件  找到theme  行
  修改theme: yilia（下载的主题）
 更新  
  cd themes/yilia
  git pull

 hexo clean  清楚一下
 hexo g         重新生成一下
 hexo s         在重新启动一下
 本地4000端口可以查看了

 hexo d 重新部署一下

片刻刷新即可-_-
